# Juego "Hundir La Flota" en Java

Repositorio que contiene todo el código fuente del juego y tabmién un ejecutable para poder jugarlo.

## Funcionamiento

Al iniciar el juego, se pide que coloques 5 barcos de diferentes tamaños en la cuadricula de abajo. Se pueden posicionar horizontalmente y también verticalmente pulsando la tecla "R" o haciendo click en el boton "Rotar". También existe una opción arriba para cambiar los colores de los barcos del jugador.

Una vez colocados los barcos del jugador, el contrincante, controlado automáticamente por el ordenador, situará sus barcos. Tras esto, el jugador tiene que ir seleccionando casillas de la cuadricula superior para tratar de hundir los barcos del contrincante. Si al elegir una casilla, el color que sale es azul, significa que no hay ningún barco en esa casilla. Si el color es amarillo, significa que has tocado parte de un barco. Cuando hundas un barco entero, las casillas que pertenezcan a ese barco se volverán de color rojo.

El juego termina cuando el jugar hunde todos los barcos del contrincante, o viceversa.
